# OpenBSD 3.4 (i386)

OpenBSD 3.4 fully boots and it can be installed.

# Configuration

Network configuration should use DHCP.

Switch shell to ksh.

export EDITOR=vim

# Components

* GCC: 2.95.3
* OpenSSH: 3.7.1
* httpd: based on Apache 1.3.26
* XFree86: 4.3.1

X11 works properly now without having to disable acceleration.
